STORY BEHIND THE PROJECT

Project Name:

COVID-19 Australia status data visualization

Project background:

During the COVID-19 pandemic crisis. People are keen to know the current situation as well as the historical data to get a better idea of the trending.

Project goal:

I have built a data visualization app consuming an API from the community and done some transformation to display the overall trending as well as the trending in each state. I also made the application public and share this information with others.

OBSTACLE & SOLUTION

Obstacle:

Once blank page happened when third party API failure.

Solution:

Build local file system cache for the data daily for the backup plan for the third party API failure.
